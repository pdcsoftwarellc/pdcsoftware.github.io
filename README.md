# Air Pong: Rivals Website

Static GitHub Pages product and support site for **Air Pong: Rivals**.

- Game landing page: `/`
- Player support: `/#support`
- Privacy policy: `/privacy/`

The site has no build step, dependencies, cookies, or analytics. GitHub Pages can
publish it directly from the repository root on the `main` branch.

The optimized artwork in `assets/` is derived from the iPhone App Store campaign
in the Air Pong app repository.
