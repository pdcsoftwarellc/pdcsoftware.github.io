# Air Pong: Rivals Website

Static GitHub Pages product and support site for **Air Pong: Rivals**.

- Game landing page: `/`
- Player support: `/#support`
- Privacy policy: `/privacy/`

The marketing page is a static Next.js export from the private Air Pong app
repository. GitHub Pages publishes the generated files directly from this
repository's `main` branch. The site has no cookies or analytics.

The latest product captures live under `screens/`. The standalone privacy page
and its supporting assets remain in this repository so App Store and player
support links stay stable across marketing-site refreshes.
